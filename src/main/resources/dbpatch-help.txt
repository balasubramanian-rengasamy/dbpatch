Goals:
	dbpatch:help 		- this help message
	dbpatch:help-parse 	- parse sql (see dbpatch.file system property)
	dbpatch:list 		- display list of patches (see patch statuses section)
	dbpatch:patch 		- patch database 
	dbpatch:interactive	- interactive mode
	dbpatch:skip-errors	- mark patches "in progress" as comited
	
System properties:
	maven.dbpatch.file		- file to parse for help-parse goal
	maven.dbpatch.configuration - selected configuration from properties file (see documentation)
	maven.dbpatch.skip		- skip dbpatch operations 
	
Patch statuses:
	* = patch is available in database
	P = patch in progress
	- = patch not available in database and should not be added
	+ = patch not available in database and should be added
	
More information at http://jsoftware.org/maven-dbpatch-plugin
